### 用于导出[你的日记](nideriji.cn)的Python脚本
------

**用法**

```cmd
python ./main.py
```

**附加参数**

| --save-diaries | 保存日记文件 ( 在.data文件夹下 ) |
| -------------- | :------------------------------- |

**示例**

```
python ./main.py --save-diaries
```
